V7 Changes
~~~~~~~~~~

NEW

- Radius of "Range Boost" activation decreased from 1 to 0.2 of turret range

- Bomb now kills any enemies in range when exploding

- Player now receives proper damage when "Shield" upgrade is active
- Special upgrade purchase buttons now deactivate only if an upgrade is purchased successfully
